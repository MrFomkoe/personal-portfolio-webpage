E-Shop React project

Project Authors

Roman Ponomarjov

Project Description

Check out websiste

The project is a web app (e-shop) prepared within the test for the React developer position.

The website allows its user to switch among categories of products and add them to the cart using the selection of the particular product's attributes. The website contains the following features:

  • Dinamically rendered components which are routed via the Reac Router.
  • GraphQL dinamic fetching of all necessary data from the server: categories, products, currencies, prices, etc.
  • The data is being stored and passed via React-Redux.
  • The product can be added through the category page or through product's dedicated page.
  • The cart can be seen as the overlay from any page or from it's dedicated page.
  • The currency can be switched at any time and the price of the product / total cart sum will be dinamically recalculated.
